使用hexo创建个人博客网站

发表于2025-05-08|更新于2025-05-08
|浏览量:

@TOC

环境准备

node安装

进入node官网安装node.js
使用node -v检查是否安装成功

安装成功后应该出现如上界面

hexo安装

使用npm install hexo -g来安装hexo
测试hexo是否安装成功
hexo -v

ssh配置

使用ssh-keygen来生成新的ssh公钥
一路回车就行了
然后进入C:/用户/你的用户名/.ssh里面

打开这个文件夹里面的东西，复制下来，前往github

使用hexo命令搭建个人博客网站

hexo命令

hexo init初始化
hexo g生成静态文件
hexo s预览，是本地的，在localhost:4000下运行可以看到效果
hexo d部署

部署到github

创建仓库

在github上创建一个仓库，格式必须按照如下创建
用户名.github.io
否则访问不成功，用户名也不要有大写

修改_config.yml文件

找到yml文件里面的deploy
替换成如下的：

然后使用hexo d部署，就可以通过用户名.github.io进行公网ip的访问了

编写博客

新建一篇名为：chenge-first-blog的博客
命令：hexo new ‘chenge-first-blog’
可以看到生成了一个chenge-first-blog.md文件，通过这个md文件去写博客

默认的内容是这样的：
title: chenge-first-blog
date: 2025-03-07 21:30:43
tags:
这里开始编写正文

编写好之后
hexo g生成静态文件
hexo d部署
就完成了一篇博客的撰写

主题扩展

博客所在目录下打开git bash，再通过Git clone https://github.com/theme-next/hexo-theme-next themes/next即可完成。
打开根目录下的_config.yml,查找theme字段，将字段改为theme: next(冒号:之后要有空格分隔，否则无效) ，之后通过hexo g和hexo s，再在浏览器中访问localhost:4000即可本地预览主题效果。
