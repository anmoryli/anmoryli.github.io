修改说明

  1. URL 判断和文件下载
    • 添加了 downloadFileFromUrl 方法，用于从 URL 下载文件并保存为本地临时文件。
    • 使用 Files.createTempFile 创建临时文件，文件后缀从 URL 中提取（如 .docx）。
    • 通过 URL.openStream() 下载文件内容，并写入临时文件。
  2. 动态路径处理
    • 在 getAiResult 中检查 content 是否以 http:// 或 https:// 开头。
    • 如果是 URL，则调用 downloadFileFromUrl 获取临时文件路径。
    • 如果是本地路径，则直接使用 Paths.get(content)。
  3. 文件验证
    • 在使用 filePath 之前，检查文件是否存在（Files.exists(filePath)）。
  4. 临时文件清理
    • 如果处理的是 URL 下载的临时文件，在分析完成后使用 Files.deleteIfExists(filePath) 删除临时文件，避免占用磁盘空间。
  5. 异常处理
    • 整个方法被包裹在 try-catch 块中，捕获所有异常并返回 Optional.empty()，避免程序崩溃。
    • 打印详细的错误信息，便于调试。
